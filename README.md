# 🚀 Taskify - Task Management System

Taskify is a full-stack task management web application that allows users to create, manage, update, and track tasks efficiently. The application includes secure user authentication using JWT, role-based access control, and a responsive user interface built with React.

---

## 📌 Features

### 🔐 Authentication & Authorization
- User Registration
- User Login
- JWT Authentication
- Password Hashing using bcrypt
- Protected Routes
- Role-Based Access Control (Admin/User)

### ✅ Task Management
- Create Tasks
- View Tasks
- Update Tasks
- Delete Tasks
- Task Status Tracking
- User-specific Task Management

### 🎨 Frontend
- Responsive UI
- Login Page
- Registration Page
- Dashboard
- Task Management Interface
- Error & Success Notifications

### ⚙️ Backend
- RESTful APIs
- Express.js Server
- MongoDB Database
- Mongoose ODM
- Middleware for Authentication & Authorization
- Centralized Error Handling
